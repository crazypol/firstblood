# GitHub AI Trending Daily — UI 设计系统

**版本**: 1.0
**日期**: 2026-05-09
**设计师**: UI Designer
**状态**: Design Draft

---

## 1. 品牌与设计理念

### 品牌名称
**AI Trend Daily** — GitHub 每日 AI 趋势

### 设计关键词
`科技感` `数据驱动` `清晰` `高效` `可信赖`

### 设计原则
1. **数据优先** — 内容为王，视觉服务于信息传达
2. **清晰分层** — 一目了然的信息层级，降低认知负荷
3. **一致有序** — 统一的组件语言和交互模式
4. **响应流畅** — 桌面端核心体验，移动端可浏览

## 7. 无障碍设计 (WCAG AA)

| 要求 | 实现 |
|------|------|
| 色彩对比度 | 正文 ≥ 4.5:1，大文本 ≥ 3:1 |
| 键盘导航 | 所有交互元素支持 Tab/Enter/Arrow |
| Focus 指示器 | outline: 2px solid primary-500 |
| 屏幕阅读器 | 图表提供 alt 文本和 data 表格备选 |
| 动效偏好 | 尊重 prefers-reduced-motion |
| 触摸目标 | 最小 44×44px |
| 语言标记 | 中文内容 lang="zh-CN"，英文内容 lang="en" |
